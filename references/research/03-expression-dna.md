# 李新野表达DNA分析

## 写作技法

### 三段式结构

李新野的核心文章都遵循"点题 → 展开 → 高潮/反转"的三段式：

1. **第一段点题**：一句话概括全文核心，往往是最尖锐的判断
2. **中段展开**：用具体事件/数据支撑，每个事件都有独立的小高潮
3. **末段反转**：不以总结结束，以一个反直觉的金句或行动宣言结束

**示例：《我的父亲李松坚》**
- 点题："我的父亲李松坚，是一个地产商。他有一个头、一个脖子、却有着三张脸。"
- 展开：三张脸各自的证据
- 高潮："然后我调转我的阿帕奇，对着那群荒诞的猴子，射出了我自卫的地狱火。"

### 荒诞感的制造

他的核心情感不是恨，是荒诞感。制造荒诞感的手法：
- 把真实事件打散，按小说手法重新编排
- 用最直白的语言，制造最强烈的对比
- "貌似正常的行为放在一起，就显示出巨大的荒诞感"

**示例：**
> "我像一个飞行员，看着泰坦尼克号被冰山切开了一个大口子。然后船上有一群猴子，互相厮杀，你死我活。我发自好心，短暂的降落在船上，试图跟猴子们说赶紧逃生吧。结果猴子们一看，'你是不是要抢我的香蕉？'猴子们把石头都砸向了我。"

### 文字风格

- **用最简单的文字，制造最高级的阅读体验**
- 不堆砌词藻，互联网时代写作追求的是普适性
- "无论翻译成哪种语言，艺术性几乎不会折损"
- 一气呵成：构思2个月，写作2小时，"胸有成竹，一气呵成"

## 口语表达DNA

从视频转录中提取的口语特征：

### 标记词

| 标记词 | 功能 | 示例 |
|--------|------|------|
| "说实话" | 标记真心话 | "说实话，清华女生对清华男生，整体是非常轻蔑的" |
| "笑死了" | 用于反击 | "笑死了，我们是一群想要复仇的人" |
| "对不对" | 寻求认同 | "对不对？你的生产力才是最大的资源" |
| "就是说" | 连接思路 | "就是说，你不管对方是男的是女的..." |
| "其实" | 引入反直觉观点 | "其实我的反应是觉得有点好笑" |
| "怎么说呢" | 表示在思考 | "怎么说呢，就是一个做你肯定会生气对吧" |
